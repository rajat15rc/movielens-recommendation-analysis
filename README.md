# MovieLens Recommendation Analysis
MovieLens recommendation analysis in Pandas, covering popularity-based recommendation, user preference analysis, movie-pair co-occurrence, and lift.
## What this project is about
This project explores the basics of recommender systems using the MovieLens small dataset and Pandas.

Rather than building a full production recommendation engine, the notebook focuses on core recommendation ideas through exploratory analysis and simple recommendation strategies.

## What the notebook covers
The workflow includes:
- loading and exploring the MovieLens dataset,
- analysing user preferences and genres,
- popularity-based recommendation,
- movie-pair co-occurrence analysis,
- lift-based association analysis for identifying strong movie pairings.

## Dataset
The notebook uses the MovieLens small dataset and works with:
- 100,836 ratings
- 9,742 movies

## Why this repo is useful
This repository is a good introduction to recommendation thinking because it moves step by step from:
1. basic data exploration,
2. simple popularity-based recommendation,
3. personalised preference analysis,
4. association-based recommendation ideas.

## Main file
- `Recommender Systems.ipynb`  
  Main notebook for all analysis and recommendation tasks.

## What a non-technical reader should take away
This is a learning-focused recommender-systems project showing how movie recommendations can be approached through:
- what is popular,
- what a user tends to like,
- and which movies tend to co-occur strongly in user histories.
